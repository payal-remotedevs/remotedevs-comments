Introduction
============

The :extension:`rd_comments` extension provides a clean, threaded comment system
for TYPO3. It is designed specifically to extend the functionality of the popular
:extension:`news` extension by Georg Ringer.

It enables site visitors to post comments and nested replies on news detail pages,
encouraging community engagement, discussion, and feedback directly on the website.
In addition to frontend functionality, the extension includes a backend module that
allows moderators to review and manage comments efficiently.

The extension is built on **Extbase** and **Fluid**, making it easy to customize,
extend, and integrate into TYPO3 projects that use :extension:`news`.

Why you should use rd_comments?
--------------------

- Ideal for blogs, magazines, or news portals using :extension:`news`
- Fully integrated with the TYPO3 backend for comment moderation
- Flexible Fluid templates for frontend customization
- Clean code structure following TYPO3 extension best practices

Key features
------------

- Seamless integration with :extension:`news`
- Support for nested (infinite-depth) replies
- Frontend form for submitting comments
- Backend module for reviewing, approving, and deleting comments, as well as viewing likes count
- Built with Extbase and Fluid
- Developer-friendly and easy to extend
